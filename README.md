Compound Interest Calculator:
This project is a web-based Compound Interest Calculator that allows users to calculate the total capital and accumulated interest over a specified period. The application uses Chart.js to visualize the results in interactive graphs.

Features:
Input fields for initial capital, interest rate, and investment duration.
Dynamic graphs displaying total capital and accumulated interest as a percentage.
Dark and light mode toggle for better user experience.

Technologies Used:
-HTML
-CSS
-JavaScript
-Chart.js

How to Use:
Enter the initial capital, interest rate, and duration in years.
Click the "Calculate" button to see the results.

Español

Calculadora de Interés Compuesto:
Este proyecto es una calculadora de interés compuesto basada en la web que permite a los usuarios calcular el capital total y el interés acumulado durante un período específico. La aplicación utiliza Chart.js para visualizar los resultados en gráficos interactivos.

Características:
Campos de entrada para capital inicial, tasa de interés y duración de la inversión.
Gráficos dinámicos que muestran el capital total y el interés acumulado como un porcentaje.
Alternar entre modo oscuro y claro para una mejor experiencia de usuario.

Tecnologías Utilizadas:
-HTML
-CSS
-JavaScript
-Chart.js

Cómo Usar:
Ingresa el capital inicial, la tasa de interés y la duración en años.
Haz clic en el botón "Calcular" para ver los resultados.
